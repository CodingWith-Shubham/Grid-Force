QUICK START
================================================================================

WINDOWS:
  start.bat

MACOS/LINUX:
  chmod +x start.sh && ./start.sh

MANUAL (ANY OS):
  Terminal 1: cd backend && npm install && npm run dev
  Terminal 2: cd frontend && npm install && npm run dev
  Browser: http://localhost:3000
